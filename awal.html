  • Gambar

    • Untuk menampilkan gambar menggunakan tag img dengan artribut src. Artribut src berarti source gambar. Gambar sebaiknya satu folder dengan file html untuk mempermudah. Jika tidak satu folder maka harus diberi "/" sebelumnya dan beri nama foldernya.(lihat source code-nya untuk lebih tau)

Ada banyak properti pada tag table bisa dilihat pada link yang ada dibawah. (lihat source code-nya untuk lebih tau)
